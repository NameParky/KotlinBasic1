
# 💡 현재 프로젝트의 목표
>1. 리니어 레이아웃을 활용하여 레이아웃을 조절할 수 있다.
>2. res를 활용하여 폰트에 색과 글꼴을 지정하여 사용할 수 있다.
>3. 이미지버튼을 생성하여 이미지를 추가시킬 수 있다.
>4. 이미지 버튼에 클릭이벤트를 부여하여 클릭하였을 때 텍스트 내용을 변경시킬 수 있다.
